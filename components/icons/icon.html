Icon ​

BsIcon is small, and a lightweight component. It can be used inside almost any component or html element. BsIcon loads icon from Google Material Icons dynamically.

Overview ​

The icon's name used by <bs-icon> is the same as in Google Material Icons. <bs-icon> has four icon variants: Filled (default), Outlined, Rounded, and Sharp. Use size property to sets the icon size, otherwise default size will be used.

API Reference ​

PropertyTypeDefaultDescription
flipStringFlip the icon, valid values are: horizontal, vertical, both.
heightNumber24The icon's height in pixel.
icon requiredStringThe icon's name. Use any valid Google Material Icons name.
pulseBooleanfalseApply pulse animation.
rotateNumberRotate the icon, valid values are: 90, 180, 270. v2.0.0
