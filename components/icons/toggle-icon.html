Toggle Icon ​

BsToggleIcon is a lightweight component which is used to toggle two different icon. It can be used inside almost any component or html element. BsToggleIcon load icon from Google Material Icons dynamically.

Overview ​

<bs-toggle-icon> use icon's name as in Google Material Icons. <bs-toggle-icon> has four icon variants: filled (default), outlined, rounded, and sharp. Use size property to sets the icon size, otherwise default size will be used.

Updated on v2.0.0

IMPORTANT

The <bs-icon-toggle> tag still works, but has been deprecated since v2.0.0.

API Reference ​
