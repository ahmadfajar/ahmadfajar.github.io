Icon Spinner ​

BsIconSpinner is an icon component that is normally used for displaying loading indicator.

Added since v2.0.0

Overview ​

<bs-icon-spinner> is typically used for displaying loading or refresh indicator. It can be used inside almost any component or html element. <bs-icon-spinner> has two built-in animations, which are: Spin and Pulse. These animation can be enabled via property spin or pulse, but do not use both property together.

API Reference ​

PropertyTypeDefaultDescription
colorStringThe component color appearance. Any MDBootstrap Color variants and Material Color variants can be used.
pulseBooleanfalseApply pulse animation.
spinBooleanfalseApply spin animation.
sizeNumber24The icon size.
