Component Catalog ​

Table of Contents on this section.

Alert
A component that provides contextual feedback messages for typical user actions with the handful of available and flexible alert messages.
Animations:
Mask Loader
A component which are typically used for displaying a loading progress with backdrop overlay.
Ripple
Small and lightweight component which is used to show ripple animation when click or touch event occured.
Transition Effect
A Transition animations effect that will make an UI feel great.
Badge
A lightweight component which can be used for adaptive tagging such as adding context to just about any content.
Buttons:
Button
A component for actions in form, dialog, and more. Includes support for a handful of color variations, sizes, states, and more.
Toggle Button
A group of buttons with toggle capabilities. It is primarily used to visualize choices with different approaches.
Card
A flexible and extensible content container. It includes sub-component for headers, footers, and content with variety background colors, and powerful display options.
Chips:
Chip
A component which is used to convey small pieces of information to create arbitrary items, like categories or tags.
Chip Group
A component which extends the BsChip's component by providing groupable functionality. It is used for creating groups of selections using chips.
Container & Navigation:
App Container
A lightweight components which is used as layout container for your application.
Appbar
A lightweight component which is placed at the top of the page as a container for page title, logo, menus, and action buttons.
List Nav
A component used for displaying a series of item and mainly used for navigation. It can contain an icon, nested items, and provides a consistent styling for organizing the items.
Side Drawer
A lightweight component's container which may contains application navigation, logo and others.
Date Picker
A component that generates a calendar style date selection widget, which can be used to create customized date picker inputs.
Dropdown Menu
A popup container which are typically used for displaying lists of links and actions in a dropdown menu style.
Icons:
Icon
Small and lightweight component which can be used inside almost any component or html element.
Icon Spinner v2.0.0
A lightweight component which is used for displaying loading indicator.
Icon SVG v2.0.0
A lightweight component that load an icon from Google Material Icons dynamically.
Toggle Icon
Small and lightweight component which is used to toggle two different icon. It can be used inside almost any component or html element.
Image:
Avatar
A component which are typically used to display a user profile as a picture, icon, or short text.
Image Holder
A lightweight component which is primarily used to create SVG shape to visualize image placeholder.
Lighbox
A modal image gallery component. It works with a series of images and includes support for previous/next controls, indicators and toolbar buttons.
Input Controls:
Text Area
A component that let users enter a large amounts of text into an UI.
Text Field
A component that let users enter text into an UI.
List Tile
A component used to display information. It can contain an image, content, actions, subheaders and much more. BsListTile represents content in such a way which makes it easy to identify a specific item in a collection.
Popover
A lightweight component which is used as a popup container.
Progress Control:
Progresss
A component which are typically used to visualize the data loading progress.
Progress Bar
Bootstrap progress bar's component featuring support for animated background and text labels which are typically used for displaying simple progress bar.
Tabs
A component which is used for hiding content behind a selectable item. They can be used as a pseudo-navigation for a page, where the tabs are links and the tab-items are the content.
Tooltip
A component which is useful for conveying information when a user hovers over an element.
Miscellaneous:
Divider
Small and lightweight component which is primarily used to separate content within lists or layouts.
Subheader
Small and lightweight component which is used as lists title.
