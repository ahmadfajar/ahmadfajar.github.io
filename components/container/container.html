App Container ​

BsApp is a lightweight components which is used for your application containment. If you intend to build an app with Vue MDBootstrap as the main UI components, and the app layouts such as Navigation Bar and SideBar will be using Vue MDBootstrap, then you need these components so that BsAppbar and BsSideDrawer components can work properly.

Updated since v2.0.11

Basic Usage ​

IMPORTANT

The <bs-app-container> tag still works, but has been deprecated since v2.0.11.
